# matman

matman 是一个用于数据模拟和段对端测试的工具。

> Node `v7.6.0` 开始默认支持 `Async/Await`，请将 Node 升级到 `v7.6.0` 以上。

## 特性

- 强大的自定义产生模拟数据的能力
- 灵活的获取不同模拟数据
- E2E（段对端）测试的集成

## 安装

```
$ npm install matman --save
```
