      # Fastest 发布日志

      # v3.5.2 (2019-11-05)

      新特性：

      1. 增加了 试一试 的功能，可以帮助快速掌握 fastest 的基本使用

      体验优化：

      1. fastest.oa.com 管理端全新改版
      2. 使用 redis 来缓存鉴权接口，接口耗时降低了 30% ；且只需打开页面第一次请求鉴权接口，后续请求沿用缓存结果，即由原来的“N次请求调用N次鉴权接口”变成了“N次请求调用1次鉴权接口”，大大优化了页面整体的加载速度

      bug修复：

      1. 修复了页面被 iframe 引入时输入 token 登录无响应的问题 #246 @nateji
      2. 修复了 https 请求问题 #248 #249
      3. 修复了 x-forwarded-host ssrf 漏洞 #237
      4. 修复了 post 请求时 content-type 被重写的问题 #247 @simingchen
      5. 修复了业务返回的 set-cookie 丢失的问题 #243 @simingchen

      # v3.5.1(2019-10-26)

      新特性：

      1. 应用管理员可以主动添加人员到应用了，入口在“应用管理”-“权限列表”中 #187 @tabbhe
      2. 管理后台新增公共功能，重要事项（例如升级等）将通过公共进行提醒

      体验优化：

      1. 优化了申请应用之后的提示
      2. 优化了 oa 登录和 token 校验的体验 @nateji

      bug修复：

      1. 修复了 post 请求在某些情况下失效的问题 #216 @nateji
      2. 修复了iOS now app 里面小面板切换环境的问题 #140 @lewischeng @kevinyyang
      3. 修复了执行 clear site data 之后可能出现的页面空白问题 #218 @kurtshen
